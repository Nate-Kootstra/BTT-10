Word Processing

The Second Unit

Word Processing is another way to say MS Word. This unit works exclusively with MS Word and no other word processing software despite what the name implies. We learn about how to use MS Word, an application that I will rarely need to use in my day to day life because I use LibreOffice for Linux. Some of the principles still apply, however a majority of the unit appears to be memorizing how to use formatting on MS Word specifically. Despite being someone who dislikes Microsoft products and prefers open source alternatives when availabe, I have learnt a few things that will be useful when using LibreOffice.
